# PreoperativeSAM
PreopreativeSAM, leveraging preopreative information to create preoperative prompt guiding SAM for intraopreative images segmentation.

## Install
create a virtual env with conda
```bash
conda create --name preSAM python==3.11.7
conda deactivate
conda activate preSAM
```

install torch
```bash
pip install torch torchvision torchaudio
```


downlosd the repository using ssh connection
```bash
git clone https://github.com/AngeloLasala/PreoperativeSAM.git
```

install the requirements
```bash
pip install -e .
```
